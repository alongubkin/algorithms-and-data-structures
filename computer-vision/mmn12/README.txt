Newer versions of OpenCV removed support for SIFT.
In order to run bow.py, install the following version of OpenCV:

    pip3 install opencv-python==3.4.2.16 --upgrade
    pip3 install opencv-contrib-python==3.4.2.16 --upgrade

Additionally, install the following package for the progress bar:

    pip3 install tqdm

Download dataset from:
http://people.csail.mit.edu/torralba/code/spatialenvelope/spatial_envelope_256x256_static_8outdoorcategories.zip
